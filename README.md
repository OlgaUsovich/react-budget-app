<!-- GETTING STARTED -->
### Launch the project

1. Clone the repo
   ```sh
   git clone https://github.com/OlgaUsovich/react-budget-app.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run project
   ```sh
   npm start
   ```
